Numero inteiro: INT = Representa numeros inteiros, como 1, -5 e 10.
Numero flutuante: FLOAT = Representa numeros com casas decimais, como 3.14, -0,5 e 2.0.
String: Representa texto ou sequencia de caracteres, como: "Olá, mundo!".
Numero Booleano: Bool = Representa valores verdadeiro (true) ou falso (false)

Estrutura de controle:
IF ou se: Condição = verdadeiro 
ELSE ou se nao: condição = falso

Estrutua de repetição - loops: 
FOR:Deve ser usado para repetir a estrutura de programação

Estrutura de de decisão - Switch:
Switch: É uma estrutura de controle que permite transferir o controle para diferentes comandos dentro do corpo do switch, com base no valor de uma expressão

Condicionadores lógicos - AND, OR e NOT:
AND: Utilizado para verificar somar as condições e o resultado da ação só acontece se ambas estiverem de acordo, ou seja, verificar se as condições são verdadeiras
simultaneamente para prosseguir.
OR: Utilizado para verificar se uma das condições são verdadeiras para prosseguir 
NOT: Utilizado para exceção, se voce quer que ele retorne com as informações exceto essa condição.
==============================================================================================
Organizando as estruturas de dados:
Array: é uma estrutura de dados que armazendo uma coleção de elementos do mesmo tipo. Os elementos dentro de um array são acessados por um indice que reperesenta
a posição do elemmento na coleção. A indexação em arrays geralmente começa em zero, ou seja, o primeiro elemento está no indice xero, o segundo elemento está
no indice um, e assim por diante.

Qual é o comando usado para adicionar todos os arquivos alterados para a área de preparação (staging)?
R: git add .

Qual é o comando usado pra criar e escrever uma mensagem de um commit?
R: git commit -m “Minha mensagem"

Qual é o comando usado para ver o histórico de commits de um projeto?
R: git log

Qual é o comando usado para verificar as informações sobre quais arquivos foram modificados,
R: git status

Qual o comando usado para adicionar apenas um arquivo alterado para a área de preparação (staging)?
R: git add meu-arquivo.txt

Qual é o comando usado para enviar as alterações locais para um repositório remoto?
R: git push

Qual é o comando usado para trazer as alterações de um repositório remoto para o seu local?
R: git pull

Qual comando é usado para clonar um repositório remoto para o seu computador local?
R: git clone

O que é um commit?
R: Um commit é quando você cria um ponto na história, ou seja, quando você registra as alterações do seu código em um repositório;
